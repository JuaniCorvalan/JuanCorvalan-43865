Buenas a todos. Mi nombre es Juan Corvalan y esta es mi primera pagina web. Se trata de un proyecto del curso de Python que da CoderHouse, en la cual utilizamos el framework Django y debiamos crear una pagina estilo blog. 

La idea principal surge de la necesidad de tener un acceso rapido y facil al contacto de alguna cancha en alguna parte del area metropolitana de Buenos Aires. Por lo que surgio dividir por zonas y en cada una de ellas cargar distintas canchas donde uno puede practicar distintos deportes. 
Norte, Sur, Este y CABA de toda la ciudad de Buenos Aires. 
Cuando se ingresa a alguna zona se veran las canchas ya ingresadas y tambien tendremos cuatro opciones:
1. Agregar una nueva
2. Modificar
3. Mirar una especificamente
4. Eliminar
5. Agregar algun comentario
Utilizando el path /aplicacion/znbuscar
                              /zobuscar
                              /zsbuscar
                              /cababuscar  podremos filtrar por localidad para que de esta forma sea 
mas facil poder encontrar la cancha que querramos.

Para poder utilizar las utilidades que brinda la pagina, es necesario crear un usuario y posteriormente loguearte. Al loguearte, tendras la opcion de agregar un avatar de identificacion como tambien la opcion de editar tu perfil con la opcion de agregar tu nombre o tu mail.
