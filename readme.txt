Mi nombre es Juan Corvalan. Decidi hacer una web estilo blog, la cual sirve para que las personas puedan encontrar
un sitio donde puedan practicar algun deporte. 
Es una interfaz sencilla, esta divido por zonas. Norte, Sur, Este y CABA de toda la ciudad de Buenos Aires. 
Cuando se ingresa a alguna zona se veran las canchas ya ingresadas y tambien tendremos cuatro opciones:
1. Agregar una nueva
2. Modificar
3. Mirar una especificamente
4. Eliminar

Utilizando el path /aplicacion/znbuscar
                              /zobuscar
                              /zsbuscar
                              /cababuscar  podremos filtrar por localidad para que de esta forma sea 
mas facil poder encontrar la cancha que querramos.

Para utilizar deberas logearte con un usuario que previamente debe ser registrado. 

Por ultimo las credenciales de adminstrador son: 
user: admin pass: admin123

Espero que sea de su agrado!
